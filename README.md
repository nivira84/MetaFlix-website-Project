# MetaFlix-website-Project

OVERVIEW
Metaflix is a free entertainment website inspired by platforms like Netflix. It allows users to explore and enjoy various categories including series, movies, and anime without any subscriptions or login requirements.
The platform aims to deliver a smooth and simple viewing experience for both PC and mobile, giving users easy navigation through pages like Home, Series, Movies, Recommended, My Library, and Feedback.

FEATURES
 - Homepage Carousel
Rotating images displayed in the index.html (Home page)
Gives a dynamic and modern look

- Interactive Movie Cards
Hover transitions showing 2 different pictures
Smooth animations when the cursor moves over movie thumbnails

- Mobile & PC Compatibility
Fully responsive layout
Designed to work on laptops, desktops, tablets, and mobile devices

- Multi-page Navigation
Includes pages for Series, Movies, Recommended, My Library, Signup, Login, and Feedback

- Simple and Clean UI
Dark-themed layout with an organized structure for easy browsing

TECHNOLOGIES USED
- HTML
- CSS
- JAVASCRIPT
